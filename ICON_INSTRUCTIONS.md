# PWA Icon Requirements

## You Need App Icons!

For your PWA to install properly, you need icons in multiple sizes. Here are your options:

---

## Option 1: Use a Free Icon Generator (EASIEST)

### PWA Builder Icon Generator
1. Go to: https://www.pwabuilder.com/imageGenerator
2. Upload a square image (ideally 512x512 or larger)
3. Click "Generate Icons"
4. Download the ZIP file
5. Extract and upload all PNG files to `/icons/` folder on your server

**What image to use:**
- Your logo
- A simple icon representing FlÃ¢nerie (walking figure, compass, etc.)
- Must be square (same width and height)
- Minimum 512x512 pixels
- PNG format with transparent background works best

---

## Option 2: Use Canva (FREE)

1. Go to Canva.com
2. Create a 512x512px design
3. Design your app icon (simple is better!)
4. Download as PNG
5. Use PWA Builder (above) to generate all sizes

---

## Option 3: Hire on Fiverr

Search for "app icon design" - usually $10-30
They'll provide all sizes you need

---

## Required Icon Sizes

You need these PNG files in an `/icons/` folder:

```
/icons/icon-72x72.png
/icons/icon-96x96.png
/icons/icon-128x128.png
/icons/icon-144x144.png
/icons/icon-152x152.png
/icons/icon-192x192.png
/icons/icon-384x384.png
/icons/icon-512x512.png
```

---

## Temporary Solution (For Testing)

If you want to test PWA before creating icons:

1. Find any square logo/image online
2. Resize to 512x512 using: https://www.iloveimg.com/resize-image
3. Use PWA Builder to generate all sizes
4. Replace with proper icons later
